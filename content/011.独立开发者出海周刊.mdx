## GPT-4o mini：速度更快，价格更低

OpenAI 最近发布了一款名为 GPT-4o-mini 的新型 AI 模型。这个模型价格比 3.5-turbo 还低，但是实测效果接近 GPT-4o。

![gpt-3.5-turbo-pricing](/assets/011/gpt-35-turbo-pricing.webp)

![gpt-4o-mini-pricing](/assets/011/gpt-4o-mini-pricing.webp)

GPT-4o-mini 输入每百万个 token 定价 15 美分，输出每百万个 token 定价 60 美分，价格降低了一个数量级。

### 性能与能力

虽然叫 “mini”，而且定位是替换掉 GPT-3.5-turbo，但它的能力可不小：

- 在多项基准测试中表现出色，优于其他小模型如 Gemini Flash 和 Claude Haiku
- 支持文本和图像处理，未来还计划支持视频和音频
- 单次可处理 128K 的上下文，输出最多 16K 个 token

实测效果，大部分能力接近 GPT-4o，远高于 GPT-3.5-turbo。

### 应用场景

GPT-4o mini 适用于多种情况，比如:

- 需要多个AI模型协作的应用
- 处理大量文本，如完整代码库或长对话历史
- 需要实时响应的客户服务聊天机器人


### 安全措施

- 预训练阶段过滤不良内容
- 训练后使用人类反馈进行调整，确保行为符合OpenAI政策
- 采用特殊的指令体系，提高对“越狱”等不当使用的抵抗力

### 可用性和定价

- 通过 Assistants API、Chat Completions API 和 Batch API 提供。
- ChatGPT 的免费、Plus 和团队用户可以访问，企业用户将在下周开始访问。
- 微调功能即将推出。

### 未来展望

OpenAI 的目标是持续降低 AI 使用成本，同时提升模型能力。他们希望未来AI能够融入每一个应用和网站，让AI技术更加普及、可靠，成为日常数字体验的一部分。GPT-4o mini是实现这一目标的重要一步。

### 个人点评

OpenAI 最近的动作都是雷声大雨点小，Claude 和 Gemini 在发布更强大模型的时候，OpenAI 却在发布更低级的模型，确实不够看的。

## 开源项目

- [next-auth-roles-template](https://github.com/mickasmt/next-auth-roles-template)

  开源 Nextjs 14 启动模板，含 Auth.js v5 实现登录授权、用户角色管理，还有使用 shadcn/ui 最新发布的图表组件实现了管理面板。

  ![next-auth-roles-template](/assets/011/next-auth-roles-template.webp)

- [gen-ui](https://github.com/bracesproul/gen-ui)

  这是一个使用 LangChain.js、AI SDK 和 Next.js 构建的 Web 应用模板，你可以基于此开发自己的 LangChain.js 和生成式应用。

  ![gen-ui](/assets/011/gen-ui.webp)

- [Profile Readme Generator](https://github.com/maurodesouza/profile-readme-generator)

  这是一个使用 Next.js 开发的创建和美化 GitHub 个人资料 README 的工具。你可以通过这个工具，快速、轻松地创建美观的的 GitHub 个人资料页。

  ![Profile Readme Generator](/assets/011/profile-readme-generator.webp)

- [Claude Dev](https://github.com/saoudrizwan/claude-dev)

  这是一个 VSCode插件，借助 Claude 3.5 Sonnet 的能力，Claude Dev 可以分解并处理复杂的软件开发任务。一般的辅助编程 AI 工具都只能完成单一的任务，但是Claude Dev 可以胜任读写文件、从头开始创建整个项目并执行终端命令。

- [Qwen2-Audio](https://github.com/QwenLM/Qwen2-Audio)

  阿里巴巴发布语音模型 Qwen2-Audio，有两大特性：1、语音聊天：用户可以自由地与Qwen2-Audio进行语音互动，而无需文本输入；2、音频分析：用户可以在互动过程中提供音频和文本指令对音频进行分析。
