## 出走 OpenAI 的 Ilya 创办安全超级智能(SSI)

以下是 Ilya 及其联合创始人为新公司发布的消息，介绍了新公司是什么、要做什么：

### 超级智能触手可及

构建安全的超级智能（SSI）是我们时代最重要的技术难题。

我们已经启动了世界上第一个直达式SSI实验室，只有一个目标和一个产品：一个安全的超级智能。

它被称为安全超级智能公司。

SSI是我们的使命、我们的名称以及我们全部的产品路线图，因为这是我们唯一的焦点。我们的团队、投资者和商业模式都旨在实现SSI。

我们将安全性和能力视为并行的技术问题，通过革命性的工程和科学突破来解决。我们计划尽可能快地推进能力，同时确保我们的安全始终领先。

这样，我们可以平和地扩展。

我们的专注意味着没有管理上的干扰或产品周期的分心，而我们的商业模式意味着安全、保障和进展都不受短期商业压力的影响。

Ilya Sutskever, Daniel Gross, Daniel Levy

2024年6月19日


## 开源项目

- [Sandbox](https://github.com/ishaan1013/sandbox)

  这是一个在线代码编辑平台，前端使用 Next.js、TailwindCSS、Shadcn UI 构建，后端使用 Cloudflare Workers、Express、Socket.io 构建，支持 AI 辅助、实时协作等功能。

  ![sandbox](/assets/007/sandbox.webp)

- [Minesweeper Online](https://github.com/zerosoul/ohminesweeper.online)

  一款使用 Next.js 和 Tailwind CSS 构建的 win98 风格的扫雷游戏。

  ![minesweeper](/assets/007/minesweeper.webp)

- [Agentic](https://github.com/transitive-bullshit/agentic)

  Agentic 是一个开源的AI函数和工具库，为AI开发者提供了一个统一的接口，使得在不同AI SDK和LLM间使用标准化AI函数变得更加简单高效。

  ![agentic](/assets/007/agentic.webp)

- [哔哩哔哩聊天机器人](https://github.com/linyiLYi/bilibot)

  由哔哩哔哩用户评论微调训练而成的本地聊天机器人，支持文字聊天，也可以生成给定问题的语音对话。

- [飞翔的狗子](https://github.com/orange2ai/dogandbone/blob/main/flying-dog-game.html)

  这是用 Claude-3.5-sonnet 写的一个小游戏，仅用了六轮对话就完成了核心功能的开发，开发演示可以看[作者推特](https://x.com/oran_ge/status/1804435039352156512)，可以来感受一下 Claude 新模型的能力。

## 工具推荐

- [namae](https://namae.dev/)

  当你启动新项目并寻找未被注册的名称时，这个工具非常有用。它能帮你检查所选名称在各个主流平台上的占用情况，确保你的名称在所有重要平台上都是独一无二的。

  ![namae](/assets/007/namae.webp)

- [Future You](https://futureyou.life/)

  这是麻省理工大学开发的 AI 聊天机器人，可以让你和60岁的自己进行对话。

  ![futureyou](/assets/007/futureyou.webp)

- [Remotion](https://www.remotion.dev/)

  Remotion 让你可以使用 React 编程来生成演示视频，平台提供了实时视频预览和快速刷新功能，让你可以精确编辑每一帧。

  ![Remotion](/assets/007/remotion.webp)

- [carbon](https://carbon.now.sh/)

  代码截图美化工具，输入代码即可使用。

  ![carbon](/assets/007/carbon.webp)

- [Open Graph Examples](https://opengraphexamples.com/)

  这个网站提供了大量的 Open Graph 图片示例，如果你不知道该怎么设计自己的 Open Graph 图片，不妨来这里找找灵感。

  ![og-examples](/assets/007/og-example.webp)
