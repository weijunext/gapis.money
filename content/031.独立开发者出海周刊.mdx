## 开源项目

- [Chrome AI Playground](https://github.com/oslook/chrome-ai-playground)

  Chrome浏览器最新版本提供了Gemini Nano的本地AI模型的，具备一些常用功能，该项目是通过浏览直接访问和体验这些功能的操作环境，推荐大家使用。Chrome AI 和 Gemini Nano 的综合平台，提供全面且用户友好的 API 支持。

  提供的功能：
    - Prompt - 交互式聊天界面用于测试提示
    - Summarize - 文本总结功能
    - Translate - 语言翻译功能
    - Detect Language - 自动语言检测
    - Writer - 人工智能驱动的写作支持
    - ReWriter - 内容重写和改进

  ![chrome-ai-playground](/assets/031/chrome-ai-playground.webp)

- [coreychiu-portfolio-template](https://github.com/iAmCorey/coreychiu-portfolio-template)

  一个很好看的 portfolio 模板，可以用来展示个人信息和产品。

  ![portfolio](/assets/031/portfolio.webp)

- [Artab](https://github.com/get-artab/artab)

  这是一个新标签页扩展，slogan是“让有史以来最好的艺术作品出现在你的新标签页”。

  插件发布一周多，就有将近 1000 个用户，说明「美」也是一种刚需。这是一个开源项目，你可以基于开源代码，做一款另一种「美」的插件。

  ![arttab](/assets/031/arttab.webp)

- [Image Background Remover](https://github.com/addyosmani/bg-remove)

  想做去除图片背景工具的朋友可以来看看这个仓库了，线上地址实测去除背景的效果不错。

  背后的原理是通过 Transformers.js 调用机器学习模型在本地处理媒体文件。因为图片不会被上传至服务器，所以从用户角度也很友好。

  ![bg-addy-ie](/assets/031/bg-addy-ie.webp)

- [Admin Dashboard UI](https://github.com/satnaing/shadcn-admin)

  一个基于 shadcn-ui 开发的管理后台，可以基于这个仓库来做自己的需求。

  ![shadcn-admin](/assets/031/shadcn-admin.webp)

- [auto-i18n-next-lingui-tiny-demo](https://github.com/HackerQED/auto-i18n-next-lingui-tiny-demo)

  lingui 一个 AI 翻译友好的 i18n 库，配合脚本做自动翻译。目前有小坑，需要和 Next.js 的版本完全匹配，要比较专业的前端知识才能排错，所以作者创建了这个 demo，这样后来人可以少走弯路，直接使用。并且 README.md 提供了完整的命令和输出供大家参考。

- [Qwen2-Audio](https://github.com/Lollipop/Qwen2-Audio)

  阿里巴巴云计算推出的大型音频语言模型，能接受各种音频信号输入，并根据语音指令进行音频分析或直接文本回复。

## 工具推荐

- [帽子云](https://www.maoziyun.com/)

  来自群友的作品，是一个 GitHub Pages / Cloudflare Pages 国产替代方案，集成化的静态网站快速创建平台，旨在解决国内用户部署网站时面临的种种困难。

  产品优点：
    - 优化亚洲地区访问线路，配备自建 CDN 边缘节点，实现极速加载。
    - 基于 etcd 构建高可用架构，提供稳定支持。
    - 集成 GitHub 自动化持续部署。
    - 内置 HTTPS，支持多环境配置。
    - 自定义域名与反向代理功能，满足多样化需求。
    - 附带流量数据分析功能，帮助开发者深入了解流量趋势。

  ![maoziyun](/assets/031/maoziyun.avif)

- [免费 AI 音效生成器](https://tiktokvoice.net/en/sounds-effect)

  免费 AI 音效生成器，适合用于小游戏开发，创意视频和广告，白噪音等场景。

  ![ai-sounds](/assets/031/ai-sounds.webp)

- [Clip a Frame](https://chromewebstore.google.com/detail/clip-a-frame/gfckafknbbnfiheldolhfohhlmdpdkpa)

  这是一个可以快捷截图视频帧的 Chrome 扩展，按下 S 键就可以把当前帧保存到剪贴板，按 Shift + S 可以保存成文件，目前支持 YouTube 和 Bilibili。

- [MCP Hub](https://www.aimcp.info/)

  针对 Claude 推出的 MCP 协议，搭建的 MCP 工具导航站。
