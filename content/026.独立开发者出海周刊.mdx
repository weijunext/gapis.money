## 开源项目

- [WR.DO](https://github.com/oiov/wr.do)

  多功能域名管理平台（三级域名分发、短链接生成、截图API），并带有管理员面板。

  ![wr.do](/assets/026/wrdo.webp)

- [kkTerminal](https://github.com/zyyzyykk/kkTerminal)

  这是一个基于 Web 的 SSH 终端连接工具，提供了图形化的终端界面，支持 SSH 连接、文件管理、中文输入等功能。该项目采用前后端分离架构，并支持 Docker 部署。

  ![kkbpro](/assets/026/kkbpro.webp)

- [nextjs-saas-template](https://github.com/jiweiyeah/nextjs-saas-template)

  一个简单易使用、seo 友好的 next.js 模板，没有复杂的操作和配置，适合快速搭建一个落地页、saas 网站等。

  ![nextjs-saas](/assets/026/nextjs-saas.webp)

- [Scraperr](https://github.com/jaypyles/Scraperr)

  这是一个自托管的网页爬虫应用，允许用户通过 XPath 指定要抓取的网页元素。用户可以提交URL和对应的抓取元素，结果以表格形式展示并支持导出Excel。

  ![scraperr](/assets/026/scraperr.webp)

- [Screenity](https://github.com/alyssaxuu/screenity)

  一款所有数据都放在本地处理的屏幕录制工具，专为工作、教育等场景设计，完全免费，无功能限制。

  ![screenity](/assets/026/screenity.gif)

- [Blinko](https://github.com/blinko-space/blinko)

  专为那些希望快速记录瞬间想法的人们设计的笔记工具，它让用户能够在灵感闪现的瞬间无缝记录想法，确保不错过任何一个创意的火花。

  ![blinko](/assets/026/blinko.webp)

- [public-portfolio-site](https://github.com/ricocc/public-portfolio-site)

  设计师的个人网站，可以用这套代码搭建自己的作品集网站。

  ![public-portfolio-site](/assets/026/public-portfolio-site.webp)

- [logocreator](github.com/Nutlope/logocreator)

  免费的开源 Logo 生成器，由 Flux 驱动。

  ![logocreator](/assets/026/logocreator.webp)


## 工具推荐

- [TempMail](https://tempmail100.com/)

  一个临时邮箱产品，适合用在临时接收验证码，支持免费使用。

  ![tempmail100](/assets/026/tempmail100.webp)

- [ShareTo](https://chromewebstore.google.com/detail/shareto/lbngllckgpcfndeneofhlkccchlkknpb)

  一站式的社交媒体分享解决方案，可以将网页上的文本、图片、链接、文章等内容一键分享到社交媒体，目前已经支持多个主流平台。

  ![shareto](/assets/026/shareto.webp)

- [Public domain vectors](https://publicdomainvectors.org/)

  免费可商用的矢量素材下载网站，包含7万张矢量图像。

  ![publicdomainvectors](/assets/026/publicdomainvectors.webp)
