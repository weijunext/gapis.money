## 抵制技术毒瘤：CSDN 与 Gitcode

  这周的头条送给 CSDN 和 Gitcode！

  最近 CSDN 旗下的代码托管平台 GitCode 被曝大规模克隆 Github 开源项目的行为，而且不仅仅克隆项目，连开源作者的所有公开信息都被复制。
  
  Gitcode 的做法包括：
    - 全盘复制项目信息：不仅复制项目代码，还将项目描述、作者信息等一并搬运至 GitCode 平台。
    - 伪造开发者主页：为被搬运项目的开发者创建主页，但这些主页并非由开发者本人控制。
    - 替换项目链接：将 CSDN 网站上已有文章中提到的 GitHub 项目链接，批量替换为 GitCode 链接。
    - 篡改项目文件：部分项目的 README 文件中包含 GitHub 地址，GitCode 甚至将这些地址也替换为了 GitCode 链接，试图阻止用户访问真正的项目地址。

  CSDN 和 Gitcode 的这套操作把一众开源作者干懵了，也有开发者直接到反馈通道问候 CSDN 祖宗了，据说比较早去问候的人的项目已经被删掉了，后来去问候的只能获得“要删除项目需要自己用 Gtihub 授权登录以获取账号权限”的回复。没想到在CSDN这里，骂娘都得赶早😮‍💨。

  不过，你如果觉得授权登录就有主动权，那就错了，他们的做法是授权登录的时候要求你同意 Gticode 读取你的 Github 隐私信息——它抄了你的公开信息，你要删除这些信息需要先上交隐私信息，这是什么天才产品设计团队！

  CSDN 和 GitCode 这种“技术大毒瘤”行径是对开源精神的践踏，也是对开发者权益的侵犯，希望开发者们坚持抵制 CSDN 和 GitCode，如果技术环境一定要变烂，也能因为我们的抵制烂得慢一点。

  [蒋涛千古](https://x.com/weijunext/status/1806147446591910073)

## 技术新闻

- [大模型2024高考](https://www.jiqizhixin.com/articles/2024-06-25)

  极客公园发起的大模型高考题测试，GPT-4o 成绩最高，字节豆包排在第二。

- [OpenAI 收购了 Multi 并将其关闭](https://multi.app/blog/multi-is-joining-openai)

  Multi 是一个多人协作平台，开发支持多人同时使用的桌面应用程序，类似于 Google Docs 或 Figma。这次并购后，Multi 将停止运营，用户数据将被删除。

- [OpenAI 又双叒叕通知封号了]()

  6 月底，OpenAI 向大量使用其 API 的个人和组织发送邮件通知，宣布将于 7 月 9 日对他们的账户进行封禁。

  封禁原因是 OpenAI 检测到这些账户的服务调用来自其不支持服务的国家，针对的是哪个国家大家都心知肚明了。

  此次封禁影响范围很大，几乎所有可能暴露真实 IP 地址的访问方式都受到了波及， 包括使用 Cloudflare 等代理服务的用户。甚至有人怀疑 Cloudflare 是此次封禁的“重灾区”，因为 OpenAI 与 Cloudflare 本身存在合作关系，Cloudflare 可能向 OpenAI 提供了相关数据。

- [Notion Sites](https://x.com/NotionHQ/status/1805619535330132425)

  Notion 发布了 Sites 功能，允许创作者通过 Notion Sites 创建个人网站、作品集、旅游指南等，该功能提供了海量的模板，对于非技术人员和不想自己搭建博客类网站的人来说非常实用，不过这是一个付费功能～。
