## 把AI作为公正第三方进行价值评估

现在我们都更加关注利用AI能力打造一些吸引眼球的产品，但是在2022年末到2023年初那段时间，更多的讨论是AI可以多大程度磨平信息差。因为根据人类的历史，每一代技术的变革，带来的都是更加公平、公正的社会，所以在大家还无法想象AI能力的时候，第一反应就是希望AI可以继续推动这一理想。

不过经过两年发展，已经很少有人提出要用AI做这样的工作了。昨天正好想整理一下「Next.js中文文档」的代码亮点，突发奇想，让AI客观总结看看，它可以作为利益无关的第三方来提供客观的价值评估。

AI做得很好，分类罗列了「Next.js中文文档」的亮点，并总结出这套模板适用于什么场景，还能得出“具有高可扩展性和定制性”的结论。

![ai](/assets/028/ai.webp)

这只是个简单的示例，如果要获得更客观且全面的价值评估，可以优化prompt让AI情感色彩更淡一些，然后提供竞品信息或行业标准，让AI有更多的思考维度，输出更完善的结果。

idea已经有了，就看有没有人能找到需要AI进行价值评估的场景了。

## 开源项目

- [infinite-canvas-tutorial](https://github.com/xiaoiver/infinite-canvas-tutorial)

  这是一个关于无限画布([Infinite Canvas](https://infinitecanvas.tools/))的教程项目。

  Infinite Canvas 有以下特性：
  - 高可扩展性 - 用户可以自由地以非线性方式组织内容
  - 缩放功能 - 模拟真实世界中的放大查看细节和缩小查看全局
  - 直接操作 - 提供基础图形的直观编辑能力,包括移动、分组和样式修改

  本仓库的目标是展示如何构建一个现代的、高性能的无限画布系统，类似于 Figma、tldraw 和 excalidraw 等产品，目前教程已更新14章。

  ![infinitecanvas](/assets/028/infinitecanvas.webp)

- [crawl4ai](https://github.com/unclecode/crawl4ai)

  Crawl4AI 是一个Python 爬虫库，完全免费和开源，性能比大部分付费服务还要好，提供 REST API 和 Python 库两种使用方式，支持 JSON、Markdown 等多种输出。

  ![crawl4ai](/assets/028/crawl4ai.webp)

- [co-op-translator](https://github.com/Azure/co-op-translator)

  Azure 基于自身 AI 能力实现的一款开源的多语言翻译工具，通过一行命令就可以实现文档的多种语言翻译。

  ![co-op-translator](/assets/028/co-op-translator.webp)

- [系统了解 Edge runtime](https://edge-runtime.vercel.app/)

  用 Next.js 的时候，很多人难以理解什么是 Edge runtime，其实它不是 Vercel 创造出来的，而是由多个组织组成的联盟（WinterCG）共同制定标准并实现的一套运行时。来系统了解一下 Edge runtime 的知识会帮助你更好地理解 Serverless 平台的运作方式。

- [Edge runtime 和 NodeJS runtime 该怎么选？](https://x.com/arvin17x/status/1839590540830269757)

  了解了 Edge runtime 的知识后，就可以来看[Arvin Xu](https://x.com/arvin17x)的完整分享了，他向我们介绍了两种运行时的区别，以及实战中如何选择运行时来规避高额账单。
