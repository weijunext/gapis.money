## GPT-4o更新了什么？

- 跨越语音、文本和视觉进行推理，效率极高，实时语音几乎零延迟
- GPT-4o 和 GPT Store将对免费用户开放
- 推出 ChatGPT 桌面端应用
- GPT-4o 的价格却大幅下调，仅为之前版本的一半

再说一个和独立开发者有关的信息，之前版本的 GPT4 对 Next.js 的 app router 训练非常有限，几乎无法用来解决 app router 的问题，但是 GPT-4o 已经完成对 app router 的学习了，独立开发者用 Next.js 开发遇到问题终于可以让 GPT 来协助解决了！

## 开源项目

- [A Single Div](https://a.singlediv.com)——只有一个Div标签，能创作出什么样的创意画面？

![asinglediv](/assets/001/asinglediv.png)

- [KawaiiLogos](https://github.com/SAWARATSUKI/KawaiiLogos)——可爱风 Logo 合集

![kawaiiLogos](/assets/001/kawaiiLogos.png)

- [react-rough-fiber](https://github.com/Bowen7/react-rough-fiber)——渲染手绘风 SVG 的 React 渲染器，可以轻松把 SVG 转换成手绘风图片

![react-rough-fiber](/assets/001/react-rough-fiber.png)

- [Strapi](https://github.com/strapi/strapi)——完全免费、采用 JavaScript/TypeScript 开发的无头内容管理系统，它拥有开箱即用的 API 和友好的管理面板，自带权限管理、默认安全、SEO 友好等特性

![strapi](/assets/001/strapi.png)

- [c-shopping-weapp](https://github.com/huanghanzhilian/c-shopping-weapp)——基于react生态，使用taro框架下，开发的完整微信小程序应用，一个精美的电商购物应用。（[@继小鹏](https://x.com/huanghanzhilian)开发）

![C-shopping-weapp](/assets/001/c-shopping-weapp.png)

- [Crawl4AI](https://crawl4ai.uccode.io/)——开源的对大语言模型友好的网页爬虫和抓取器（[@meng shao](https://x.com/shao__meng/status/1788922127498027431)推荐）

![crawl4AI](/assets/001/crawl4AI.png)

- [IMaker - 创意性的封面工具](https://github.com/slince-zero/IMaker)——这是一款用来设计封面的工具，比如你可以为你的博客、视频、公众号等设计你自己喜欢的封面

![imaker](/assets/001/imaker.png)
