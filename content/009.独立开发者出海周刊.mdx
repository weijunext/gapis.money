## 基于不同平台开发，算抄袭吗？

吉光卡片 app 的开发者[张小吉](https://x.com/hixiaoji/status/1808669590895145256)在线锤人，原因是有人在 web 端致敬了吉光卡片的功能，网站名称也跟「吉光卡片」相似，而且致敬方的开发者还到张小吉的推特、即刻和微信引流。

先介绍一下两个产品的区别：

- 吉光卡片是一款 iOS 原生应用，核心功能收费
- 致敬版是一个 web 应用，在网页端实现了几乎一样的功能，UI 像素级模仿，功能免费

争议的问题有两个：

- 抄袭
- 到被抄袭一方的评论区推广

有话题就有讨论，这场致敬与锤人的也引起了一场不大不小的争论，主流观点有两种：

[大帅老猿](https://x.com/ezshine/status/1809033729714835940)：

我第一反应，在我的概念里，这不算抄袭。我想起十几年前，尤雨溪用 HTML5 API 像素级“抄袭”了一款 iOS 原生应用 Clear 的事情，这两件事之间有哪些不同点呢？

在线版 photoshop 用和 adobe photoshop 一模一样的 UI 和用户体验获得了市场的一片好评。当然，它不可能获得 adobe 以及一众同类厂商的好评。

我过去很长时间都是在广告公司里做程序员，耳濡目染过很多市场营销方面的知识，评论区捞人截流也许在广大程序员看来不耻，但也是一种常见的同质化市场下的竞争手段。更为恶心的Apple和Win大战，可口可乐和百事大战，相信大家都听过看过。

[9YearFish](https://x.com/9yearfish/status/1809408032713109733)：

我们看一个产品的做得非常好，我们去了解他的业务，了解他的模式，再做一个类似的。这叫做借鉴。

我们看一个产品做的好，做一个跟他一模一样的，甚至到别人的用户群里面去捞人。这叫抄袭+恶心。

## 开源项目

- [meet-libai](https://github.com/BinNong/meet-libai)

  本项目旨在通过构建李白知识图谱，结合大模型训练出专业的AI智能体，以生成式对话应用的形式，让现代人可以和李白进行对话。

  ![meet-libai](/assets/009/meet-libai.webp)

- [react-tweet](https://github.com/vercel/react-tweet)

  无需使用 Twitter API，通过 react-tweet 你就可以在 React 应用中嵌入推文卡片。使用非常方便，只需要传入推文 id 就可以。

- [Removerized](https://github.com/yossTheDev/removerized)

  使用 AI 去除图像背景，免费且无限制，支持离线操作。

  ![Removerized](/assets/009/removerized.webp)

- [Avatars](https://github.com/alohe/avatars)

  这个仓库收集了来自网络的各种免费头像图片，可以用于项目里生成用户头像。

  ![Avatars](/assets/009/avatars.webp)

- [Shot Easy](https://github.com/CH563/shot-easy-website)

  开源、免费的在线图片处理工具，不但可以快速截图，还是对截图美化编辑。

  ![Shot Easy](/assets/009/shoteasy.webp)

- [海棠诗社开源](https://github.com/javayhu/haitang)

  这是一个使用 Astro 搭建的诗词网站，网站做得很好，对于想学习 Astro 的朋友来说是一个很不错的上手项目。

  ![海棠诗社](/assets/003/haitan.webp)


## 工具推荐

- [ChromeAI](https://chromeai.org/)

  Chrome 开发者版集成了 Gemini Nano，支持离线使用内置 AI 模型，ChromeAI 基于这个特性开发了与 Chrome 内置 AI 对话的平台。因为谷歌只在开发者版的 Chrome 上开放这个特性，所以前置配置比较麻烦，如果不懂配置可以参考[作者推特](https://x.com/CZXNew/status/1809058933304406197)。

  ![Chrome AI](/assets/009/chromeai.webp)

- [Midjourney Sref Hunter](https://srefhunter.top)

  Midjourney prompt 支持输入 sref 参数, 用来生成特定风格的图片。这个网站汇集许多 sref 参数的 Midjourney 案例，为 Midjourney 爱好者更方便地找到自己需要的图片。

  ![srefhunter](/assets/009/srefhunter.webp)

- [Hunt Screens](https://huntscreens.com/)

  如果你也觉得 ProductHunt 的列表不够直观，就可以来用这个网站，它给每个 ph 上的新项目都截了图，每天快速刷一下，一眼就能看出是不是自己的菜，再也不用读一堆文字点来点去了。开发者[@代码家](https://x.com/daimajia)，目前网站也已经[开源](https://github.com/daimajia/huntscreens)。

  ![huntscreens](/assets/009/huntscreens.webp)

- [AI 字幕生成工具](https://subtitles.fframes.studio/)

  将视频文件拖到这个网站里，网站就会利用 AI 处理并生成字幕，所有处理都在本地运行，不会向服务器发送任何内容。

  ![subtitles](/assets/009/subtitles.webp)

- [TrendingKeywords](https://trendingkeywords.net/)

  这是一个帮助挖掘热门关键词的网站，可以用来发现热门关键词、分析关键词趋势和挖掘关键词应用场景。

  ![trendingKeywords](/assets/009/trendingkeywords.webp)

- [批量查询域名是否注册](https://query.domains/)

  输入你想要的域名，就可以看到所有的域名后缀是否已被注册。

  ![querydomains](/assets/009/querydomains.webp)
